Nimbusのアスペクト指向サービスにおけるサーブレットインターセプタサービスを提供します。

NimbusアスペクトのInterceptorの中でも、サーブレット層に特化したInterceptorを提供します。
従来のJ2EEのサーブレットフィルタでは、web.xmlで初期化パラメータを渡す程度しかできず、フィルタに細かな設定を行う事が困難でした。
Nimbusでは、InterceptorChainCallFilterを使って、 サーブレットフィルタの役割をNimbusアスペクトのInterceptorに委譲する事で、多機能なインジェクション、Nimbusサービスとの親和性を得る事ができます。
このパッケージのInterceptorは、InterceptorChainCallFilterを使って、サーブレットフィルタに挟み込まれる事を前提としており、抽象サービスServletFilterInterceptorServiceを継承しています。
そのため、サーブレットフィルタと同様に、インターセプタ毎にリクエストパスに対してマッピングする事が可能です。 更に、サーブレットフィルタのような中途半端な正規表現ではなく、java.util.regexを使った正規表現によるパスマッピングが可能です。
