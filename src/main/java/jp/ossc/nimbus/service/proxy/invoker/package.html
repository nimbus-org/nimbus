Nimbusのプロキシサービス用のInvokerサービスを提供します。

プロキシのInvokerとして、以下のものを用意してます。

  • RMIを経由してサーバプロキシを呼び出すRemoteClientMethodCallInvokerService
  • EJBを経由してサーバプロキシを呼び出すRemoteClientEJBCallInvokerService
  • JMXを経由してMBeanを呼び出すJMXClientRMICallInvokerService
  • ローカルのServiceManagerを経由してローカルのサービスを呼び出すLocalClientMethodCallInvokerService
